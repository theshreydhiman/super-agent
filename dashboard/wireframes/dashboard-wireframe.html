Key Design Decisions

1. Subtle depth — white/alpha borders instead of shadows for a clean, layered feel
2. Muted accents — indigo primary avoids the overused blue; softer status colors
3. Consistent iconography — rounded icon wrappers with tinted backgrounds
4. Data density — compact tables & activity feeds maximize information without clutter
5. Two-column dashboard — table + activity/performance sidebar for better scanning
6. Sticky glassmorphic topbar — subtle blur keeps context while scrolling
7. No emojis in UI — replaced with styled icon wrappers (will use Lucide React icons)
